# Стартовый проект с gulp4

## Команды

```bash
npm start            # запуск сервера разработки
npm run build        # сборка без запуска сервера разработки
```

## Структура

```bash
build/               # Папка сборки
src/                 # Исходники
  fonts/             # Шрифты
  icons/             # SVG иконки
  img/               # Доп. изображения
  pages/             # Страницы проекта
  pug/               # Служебные pug-файлы
  sass/              # Служебные стилевые файлы
```
